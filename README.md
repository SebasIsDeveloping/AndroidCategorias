# Proyecto Final — Aplicación Multiplataforma con Avalonia

## Desarrollo de Interfaces — 2º DAM

### Descripción del proyecto
Este proyecto ha sido desarrollado por **Sebastián Naranjo** y **Vicente Martínez-Campos** como trabajo final para la asignatura **Desarrollo de Interfaces** del ciclo formativo **Desarrollo de Aplicaciones Multiplataforma (2º DAM)**.

El objetivo principal del proyecto es poner en práctica el desarrollo de aplicaciones multiplataforma utilizando **Avalonia**, creando una aplicación interactiva compuesta por dos minijuegos diseñados principalmente para dispositivos Android.

La aplicación explora diferentes formas de interacción con el usuario, incluyendo **gestos táctiles y reconocimiento de voz**.

---

### Minijuegos incluidos

**1. Juego de arrastrar y soltar (Drag & Drop)**  
En este minijuego el usuario debe **arrastrar diferentes platos a su categoría correspondiente**.  
El objetivo es clasificar correctamente los platos según el tipo de comida al que pertenecen.

**2. Juego con reconocimiento de voz**  
Este minijuego se divide en **dos etapas**:

- **Primera etapa:** el usuario debe **decir mediante voz el ingrediente que falta en un plato** mostrado en pantalla.
- **Segunda etapa:** después, el usuario debe **decir a qué categoría pertenece ese plato** utilizando también reconocimiento de voz.

Este minijuego utiliza las capacidades de **reconocimiento de voz del dispositivo** para validar las respuestas del usuario.

---

### Tecnologías utilizadas

- Avalonia UI
- .NET
- C#
- Reconocimiento de voz
- Eventos de interacción táctil (Drag & Drop)

---

### Plataforma objetivo

La aplicación ha sido diseñada principalmente para ejecutarse en:

Aunque, gracias a Avalonia, puede adaptarse a otras plataformas compatibles.
